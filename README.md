# Coffee Brew Calculator

A free online coffee brewing calculator that helps you calculate the perfect coffee-to-water ratio for various brewing methods.

## Features

- **Multiple Brewing Methods**: Pour Over, French Press, Espresso, Cold Brew, AeroPress
- **Adjustable Ratios**: Fine-tune your coffee-to-water ratio
- **Strength Preference**: Light, Medium, or Strong options
- **Unit Conversion**: Switch between Metric (g/ml) and Imperial (oz)
- **Brewing Timer**: Built-in timer with audio notifications
- **Save Recipes**: Store your favorite recipes locally

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Build

```bash
npm run build
```

The static export will be in the `out` directory.
